论文解读之Deep Residual Learning for Image Recognition‘

Posted on 2018-11-25

摘要：
深度神经网络很难训练，所以我们提出了一个残差网络。对于传统的网络，虽然理论上越深表达能力越强，但是实际上由于难以训练，越深反而表现越不好。为了解决这个问题，以及梯度消失，我们提出残差网，而且在各大比赛中，我们成绩优异。
介绍：
深度网络一层层分析聚集处不同层次的信息。最后进过分类器的处理。最近的研究也表明深度对于一个网络很重要。网络越深越好么？会有梯度爆炸或者消失的问题，不过这个通过BN可以解决。但更深的网络被训练时，还会有退化的问题。就是网络一点点变深，准确率先是提高，然后饱和。然后下降。这个退化还不是由于过拟合导致的，因为在训练集上的错误率就上升了。
一个对比实验是，一个浅层网络，和与之配对的深层网络，深层网络就是在浅层的基础上加了几层恒等映射，理论上深层网络不会比浅层差，但是事实上就是有时会差。
我们提出的残差网络是这样的：我们希望深度堆叠的非线性网络学习一个残差表达，而不是直接学习潜在映射。也就是深度网络学习一个改变的变化。最终映射由恒等映射和残差映射组合而成。我们觉得这个残差网络更好优化，比如说，如果我们最终期望的目标就是一个恒等映射，那我们只需要把残差网络那边参数全部学为零。这比让一个单纯的深度网络学习一个恒等映射简单的多。而且我们这个残差网络深的越深表现越好，这也给更深的网络设计提供的可行性。（原来太深了反而训练不好）残差单元如下图：

注意，这个skip connection需要跳至少两块，应为一块的话其实和给滤镜层参数加一没区别。
相关工作：
残差表达：大概就是说残差表达比直接映射好优化还是效率高啥的。
short connection：有相关的短连接获得了成功，我们的短连接也没有引入新的更多的参数和计算复杂度。与我们工作同时进行的还有highway network。

我们学习的是一个残差映射：比如原来映射是H(x),现在我们学习的是H(x)-x;因为我们通过一些深度加深学习退化的现象猜想stack nultipul nonlinear layers表达恒等映射困难。

论文解读之Exposure: A White-Box Photo Post-Processing Framework

Posted on 2018-11-19 | Edited on 2018-11-20

0摘要：
图像的修复很重要。但是得到监督学习的成对图片很难。我们提出的方法可以使用非成对的图片，并且输出修复的图片和一系列有意义的修复操作。我们的系统使用深度学习网络学习如何在原图上执行一些列修复操作。这是通过建立一系列模型来模拟不同修复操作，然后投入网络训练而达到的。为了用正确的滤镜参数和顺序，我们使用了一个深度加强学习网络去学习下一步干什么。
1介绍：
人们对图片的喜好各有不同，所以图像修复的时候应该关注个人喜好，那么这个好的图片的参考可以来自这个人在网上找的图片或者他喜欢的摄影师的作品。这样才能生存最适合他自己的图片增强方式。
2我们的主要贡献：
一个端到端的网络和一系列可导的滤镜
在训练网络的过程，我们的网络可以产生一系列有意义的操作过程。
用GAN产生的非监督性学习网络
相关工作：
传统/深度学习总概括/我们学习的是一系列操作，并且用unpaired的图片。与我们有关的工作是 通过将输入图片语义匹配到一个大的数据库中，寻找到颜色和色调风格与输入图片相关的示例图片，然后把风格化后的图片映射到适合的色彩与亮度分布中去。如果参考图片选择的不当，这种方法会产生很奇怪的颜色。而我们的办法是学习总体风格，而不是单一图片的统计学数据。
一个叫Aperture的软件也可以自动修图，这个软件修饰提供一系列操作，用户可以更改。我们的系统也差不多，特别的，我们的工作解释了了深度神经网络下的潜在操作。
基于数据的图片生成：纹理合成，超分，都是通过映射数据库中的一小块图片做到的。在生成一种固定种类的图片任务中，自动编码机是很实用的。（learn a compressed latent)
GAN: 最初的gan就是从一个潜在的空间去映射到目标数据的分布。后来GAN有一个变体，就是输入原图对图片有一个条件约束。然后是cyclyGAN，就是鼓励编码解码的网络所输出的可以和另一个逆向网络相互反响操作，还原。还有adversarial inverse Graphics Network.
我们的系统也用了一种条件GAN，不同的是，它产生的不是一个图片，而是一系列滤镜的参数。应为滤镜的设计是保留原图片的，这种方法保持了原图的语义信息和空间信息。而且这个滤镜是分辨率独立的，所以可以apply到任意size的图片上。我们这个工作的各个滤镜参数是一体化优化的？
强化学习：RL提供做决策的模型和智能体（agent）与环境互动的模型。在我们的工作中，一系列的滤镜序列被建模成一系列决策制定问题，还有一个有GAN中discriminator扮演的图片质量评测器，充当环境。
3模型
与传统的一次性修复不同，更易于理解的是用多种不同操作一步步修复图片。对于人类来说也是这样，一步步修复，下一步基于前一步的结果。

因此模型也是，有一个函数将输入映射到一个输出，这个输出随后作为下一次的输入，再被一个函数映射。所求的就是累积的最大奖励，对应的变化策略。如原文中公式1所示。每次映射变化的评价奖励可能有不同权重。为了把我们的问题映射到RL框架中，我们把步骤分成两块。第一，选择一个离散的滤镜，第二，选择这个滤镜的连续性函数。策略同样也分成两块，一个是把状态输入，输出各种滤镜的概率分布，（比如决定选a1 filter）另一个是输入初状态和滤镜，产生输出。由于取样一个连续变量十分有挑战性，我们采用最近的实践处理连续滤镜参数决定性的。详见后文。

5.2策略网络的训练
用policy gradient 方法训练。由于策略中有两部分需要做决策，即滤镜和参数。它们是交错学习的。
对于滤镜的选择来说这个偏导数难以计算，所以我们采用policy gradient theorem 去得到一个与滤镜选择相关的累积奖励的无偏Monte Carlo估计。而对于滤镜参数优化，我们用deterministic policy gradient theorem。
对于奖励来说，我们设定奖励应该是一个逐渐提高的质量分数加上出发机制。对于奖励有一个权重殷泽，我们设为1.并且总的来说为输入图片设置五次修改。这个次数是平和计算量和模型简洁性的。此外，我们设定固定的修改步数为了让模型更稳定（相比起让他学习自己停止修改
5.3通过对抗网络实现的质量评估
本文中，我们用WGAN。这个网络营的事Earth Mover‘s Distance 来估计两种概率分布之间的差异。图片的评分就是鉴别网络中图像的分数。这个可以这样理解，鉴别网络想给原图高分，给生成图低分，而生成图就要要得到高分。
5.4训练策略
如何平衡是优化滤镜参数呢，还是选另一个潜能可能更大的滤镜呢？我们的策略是惩罚滤镜选择，如果滤镜选择的提议过于集中，我们的惩罚就大一点。也就是奖励就比原来小一点。此外，我们发现agent又可能会连续两步用同一个滤镜，为了避免这种操作我们记住上一次滤镜的操作。这样也有利于让智能体去多尝试不同滤镜。

摘要：
我们介绍了一种增强版生成对抗网络（EnhanceGAN），以用来做图像增强。传统的图像增强一般都是训练一个强监督学习模型。这种模型的数据准备很复杂，需要把对齐的好图片和坏图片成堆匹配。而我们提出的这个网络只需要 weak supervision （只需要标明这个图片的美学质量）本文的图像增强是基于美学因素的。特别的，我们对比了一种分段式的色彩增强模型，和一种基于EnhanceGAN扩展模型学习的深度美学滤镜。我们的增强操作全部是可导的，所以可以端到端训练。然后我们这个网络还可以学习剪裁图片。衡量标准还是一些数字定义和让人们区分美术家增强的图片更好看还是网络增强的好看。

摘要：
本文讲了一种像素级别的语音识别分割网络。这个网络包含编码，解码部分和一个像素点的分类层。从拓扑学角度看，这个编码器的结构和VGG16的前13层一样。解码器的主要功能在于把低分辨率的以编码的特征映射回原分辨率的图片以获得像素级别的分割。这个SegNet的创新带你在于解码器用的是对应编码器的最大池化层的参数来计算的上采样数据。这样消除了学习上采样的必要。这个有位置索引构成的稀疏上采样图随后被用可训练的卷积滤镜去产生稠密的特征图。我们比较了我们的结构和FCN，DeconvNet等知名结构。这些比较也阐释了好多分割性能与内存之间的权衡。
SegNet主要是用来做场景理解的。所以它需要在推断时做到快速和内存经济化。这个网络相比起其他网络来说，需要训练的参数明显变少。而且可以采用随机梯度下降法端对端训练。最后我们用数据论证实验结果。
介绍：
语音分割应用范围很广，比如场景理解，推断物体之间的支持关系以及自动驾驶。原来的基于低层级的视觉线索的语音理解很快被机器学习所替代。但是如果直接把深度学习的框架套在像素点的分割上，虽然结果不错，但是边缘很粗糙。我们之所以设计这恶搞网络就是为了更好的把低分辨率特征反映射给原图，这样做更好的像素分类分割。最初呢，这个网络的工作主要是是由街道场景理解应用所促进的。这就要求理解街道场景，产生光滑的分割，保持边缘信息。并且内存和时间上在合理范围。
这个网络用encoder等的indices而不是全部数据的好处主要有：提高边缘准确性，减少参数，be incorporated easily.
这篇文章还分析了SegNet网络和FCN网络，这样来说明一下分割准确度和系统设计之间的权衡。然后分析了最近流行的各种网络结构。总之这些结构参数太多，形形色色的训练方法。

摘要：
我们现在很多照片都是有手机得到的，虽然手机拍照传感器等在不断升级，但是和专业数码相机还是有差距。本文就是通过残差卷积网络来提供一个提高图像质量的算法。我们的损失函数不是用的标准均方误差，而是自定义了四方面：上下文，颜色，纹理，和平滑度。
介绍：
就是说由于硬件差异，手机拍的和DSLR的没法比。然后说虽然现在有很多自动图像增强算法，但是它们大多是关注域全局参数调节，比如全局调节对比度，明度。并没有提高纹理质量或事把照片信息考虑在内。此外，这些算法通常是基于一个先验规则，并没有考虑到不同设备之间的差异。
相关工作：
图像增强这一大问题在计算机视觉上没有很完美的解决，但是有很多相关的子问题现在有了不错的解决：图片分辨率增强(image super-resolution)，图片去雾化(image deblurring/dehazing)，去噪(image denoising/sparse inpainting)，图片上色(image colorization)，图片调整（image color/contrast/exposure adjustment).原文中有这些方面文章的简要报告，我就不写了。
主要贡献：
1）end-to-end的神经网络可用于增强图片质量
2）6K张图片的数据库，这个数据库是由三张低质量的手机拍摄的图片和一张高质量的相机拍摄的图片组成的（后面提了一下这个数据库怎么来的，就是传感器远程控制四台设备同时同地点拍照，然后那些角度不同的再用算法对齐一下，然后还有数据增强，他们把图片分成了3100100的小块，并且说了他们试了再大也没什么用而且浪费计算资源）
3）奇妙的合成损失函数
4）成果对比
损失函数设计：
图片增强任务中，输入和目标相片不匹配是个很大问题，就是由于不同光学设备会给图片造成不同的非线性位移，所以一个个像素很难完全对齐，而且逐个像素检验差值这种方法对于视觉感觉质量上是否合适也不好说。所以本文提出了三个独立的衡量指标。分别是颜色质量，纹理，和上下文。我们现在为其独立设计损失函数，并且保证这些损失函数局部位移不变性。
1:颜色损失
这个颜色损失的计算是将临域内像素高斯模糊后的值算MSE，也就是相当于在卷积网络后面加一个固定参数的高斯核然后算均方差。
2:纹理损失
采用的另一个网络，就是生成对抗网络鉴别生成图片和目标图片之间的差距，作为损失函数
3：上下文损失
没有一个个像素比较，而是采用特征相似度比较。也就是用VGG-19这个CNN中激活函数的值差做衡量标准
4:平滑度损失（TV loss）
就是全局相片梯度变化损失，这个权重很小，所以不会伤害到高频边缘，而且有利于去掉椒盐噪声
最后这些损失被给予不同权重
